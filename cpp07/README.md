# Módulo CPP07 - Programación Genérica (Templates)

¡Bienvenido al módulo CPP07! Aquí es donde C++ muestra su verdadera potencia para la reutilización de código. He aprendido a escribir funciones y clases "universales" que no dependen de un tipo de dato específico.

Los templates son la base de la Standard Template Library (STL), y entenderlos es fundamental para cualquier programador de C++.

## 📖 Documentación

He organizado la información en dos guías principales:

*   [**Guía de Usuario (USER_DOC.md)**](./USER_DOC.md): Instrucciones para compilar y ejecutar las pruebas de funciones genéricas y mi propia implementación de un array dinámico.
*   [**Documentación de Desarrollo (DEV_DOC.md)**](./DEV_DOC.md): Un análisis técnico sobre cómo he implementado las plantillas, la gestión de memoria genérica y los retos de C++98.

## 📂 Estructura del Módulo

*   **ex00**: Funciones `swap`, `min` y `max` para cualquier tipo.
*   **ex01**: Función `iter` para procesar arrays de forma genérica.
*   **ex02**: Implementación de una clase `Array` dinámica mediante templates.
