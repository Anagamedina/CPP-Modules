# Módulo CPP08 - STL (Standard Template Library)

¡Bienvenido al módulo CPP08! Este es el punto donde C++ se vuelve realmente productivo. En lugar de construir cada estructura de datos desde cero, aquí he aprendido a dominar la biblioteca estándar para escribir código más rápido, más seguro y mucho más eficiente.

La STL es el corazón del C++ moderno, y este módulo ha sido clave para entender cómo funcionan los contenedores y algoritmos por dentro.

## 📖 Documentación

He dividido la información en dos guías principales:

*   [**Guía de Usuario (USER_DOC.md)**](./USER_DOC.md): Instrucciones para compilar y ejecutar las pruebas de búsqueda, cálculo de rangos y el stack "mutante".
*   [**Documentación de Desarrollo (DEV_DOC.md)**](./DEV_DOC.md): Un análisis técnico sobre el uso de algoritmos de la STL, la eficiencia de los contenedores y cómo extender las funcionalidades de las clases estándar.

## 📂 Estructura del Módulo

*   **ex00**: Búsqueda genérica con `easyfind`.
*   **ex01**: Cálculo de distancias mínimas y máximas con la clase `Span`.
*   **ex02**: Creación de un stack iterable con `MutantStack`.
