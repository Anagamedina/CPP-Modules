# Módulo CPP09 - STL en el Mundo Real

¡Bienvenido al módulo final de la piscina de C++! En el módulo CPP09, he puesto en práctica todo el conocimiento acumulado sobre la Standard Template Library para resolver un problema práctico: el cálculo de valores históricos de Bitcoin.

Este módulo representa la culminación de mi aprendizaje, donde la eficiencia y la robustez del código son los objetivos principales.

## 📖 Documentación

He organizado la información en dos guías principales:

*   [**Guía de Usuario (USER_DOC.md)**](./USER_DOC.md): Instrucciones para compilar y ejecutar el conversor de Bitcoin (`btc`).
*   [**Documentación de Desarrollo (DEV_DOC.md)**](./DEV_DOC.md): Un análisis técnico sobre la elección del contenedor `std::map`, la lógica de búsqueda de fechas y el manejo de errores en el procesamiento de datos.
